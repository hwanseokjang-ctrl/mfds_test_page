## LLM 프로세스 단계

<!--
전처리 > 세분화 분석 > 최종 분석 > 취합/결론 중 이 페이지에서 실제로 사용하는 단계만 나열.
세분화/최종 분석은 생략 가능. 각 단계는 config.py 의 ProcessStep 한 개로 매핑됩니다.
-->

1. **전처리** — 입력 텍스트 정제.
2. **세분화 분석** — (필요 시) 하위 분석 N종.
3. **최종 분석** — (필요 시) 통합 판단.
4. **취합/결론** — 최종 결론 텍스트 생성.

## 결과 구조

<!--
intermediates 에 담길 key 들과 final_text 형식을 설명.
-->

- `intermediates`:
  - `preprocess`: 전처리 결과 (dict/str)
  - `seg_*`: 세분화 단계별 결과
  - `final`: 최종 분석 결과
- `final_text`: 결론 텍스트 (Markdown 허용)
